# Text Collection

## Prerequisites
- [General workshop prerequisites](../WORKSHOPS.md#Prerequisites)
- install Audio Project Manager on a computer or phone
- set up an account
- get access to a project

## Goals
- learn to navigate the system
- learn to play and record
- listen to first prompt
- record whole (3m) text
- approve of parsing
- record careful speech by clause
- transcribe to another language by clause

## Follow-up Action Items
- Send your database to your consultant.
